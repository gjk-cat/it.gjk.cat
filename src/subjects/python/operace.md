# Operace

### +, - : sčítání, odčítání
```python
>>> 2 + 3
5
>>> 2 - 3
-1
```
### \*, /, // : násobení, dělení
```python
>>> 2 * 3
6
>>> 2 / 3 # vraci float
0.6666666666666666
>>> 2 // 3 # vraci int; deleni beze zbytku
0
```
### \*\* : umocňování
```python
>>> 2 ** 3
8
>>> 2 ** (1/2) # muzeme umocnovat floatem; vychazi odmocnina ze dvou
1.4142135623730951
```
### % : modulo
Modulo je operace která vrací zbytek z dělení, hodí se na zjišťování dělitelnosti čísel (viz. `sude_nebo_liche.py`).
```python
>>> 17 % 2
1
>>> 17 % 9
8
```
