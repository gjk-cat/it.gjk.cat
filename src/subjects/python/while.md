# While

Bere booleanovský výraz jako argument.
Loopuje dokud je výraz `True`.

```python
while True:
	print('tohle bezi donekonecna')
```

```python
i = 0
while i < 10
	print(i) # tohle se provede 10krat
	i += 1
```
